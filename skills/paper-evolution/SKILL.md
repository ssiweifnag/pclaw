## 核心邏輯

### 1. 文獻補充邏輯

```
輸入：當前章節 / 研究主題 / 現有文獻列表
流程：
  1. 分析研究缺口（Research Gap）
  2. 識別關鍵詞（中文 + 英文）
  3. 搜尋相關文獻（arXiv / Google Scholar / Scopus）
  4. 評估文獻相關性（主題契合度、發布年份、引用次數）
  5. 過濾低相關性文獻（<3 個關鍵詞匹配）
  6. 生成補充建議（APA 格式引用）
輸出：
  - 新文獻清單（10-15 篇，推薦優先）
  - Gap 分析摘要
  - 建議插入位置
```

**文獻來源優先順序：**
| 優先級 | 來源 | 用途 |
|--------|------|------|
| 1 | arXiv | 最新 preprint |
| 2 | Google Scholar | 學術論文 |
| 3 | Semantic Scholar | 引用網絡分析 |
| 4 | ASHRAE / ScienceDirect | 建築/IAQ 專業 |

**關鍵字擴展（IAQ + BEMS 領域）：**
```
indoor air quality + ventilation + CO2 control
demand control ventilation + IAQ + energy
BEMS + machine learning + occupancy
ASHRAE 62.1 + ventilation + Taiwan
building energy + retrofit + IAQ
```

### 2. APA 第7版核查邏輯

```
輸入：論文內容（Word / Markdown / 純文字）
流程：
  1. 檢測引用格式
     - 文中引用：(Author, Year) 或 (Author et al., Year)
     - 多人引用：et al. 使用條件（3+ 作者）
  2. 檢測參考文獻格式
     - Author, A. A. (Year). Title. Publisher.
     - DOI 格式：https://doi.org/xxxx
     - 期刊卷期頁碼
  3. 檢測常見錯誤
     - & 符號使用（英文）/ 和（中文）混用
     - 斜體標示（期刊名、書名）
     - 縮寫（et al., vs., etc.）
  4. 生成錯誤報告
輸出：
  - 錯誤清單（行號 + 錯誤類型 + 建議修正）
  - 修正後版本（可選）
```

**APA 7 常見錯誤檢查表：**
| 錯誤類型 | 範例 | 正確格式 |
|----------|------|----------|
| 作者數量 | (王, 2020) | (Wang, 2020) — 需羅馬化和姓氏 |
| 年份 | (王, 不正確) | (Wang, 2020) — 需核對原文 |
| et al. 濫用 | 2 作者用 et al. | 2 作者：Wang & Lee, 2020 |
| DOI 格式 | DOI: 12345 | https://doi.org/10.xxxx/xxxxx |
| 期刊名斜體 | Indoor Air | *Indoor Air* |
| 頁碼格式 | pp. 10-20 | p. 10–20（單頁 p.，範圍 –）|

### 3. 章節擴展邏輯

```
輸入：現有章節 / 目標字數 / 擴展方向
流程：
  1. 分析現有內容結構
     - 核心論點數量
     - 論據充分性
     - 邏輯連貫性
  2. 識別擴展機會
     - 文獻對比（其他研究怎麼做）
     - 案例補充（實際應用場景）
     - 理論深化（背後機制解釋）
     - 方法論說明（實驗設計細節）
  3. 生成擴展建議
     - 建議插入位置
     - 建議內容大綱
     - 預估增加字數
  4. 執行擴展（可選）
輸出：
  - 擴展建議清單
  - 擴展後目標字數
  - 新增內容草稿（可選）
```

**擴展方向模板：**
| 章節類型 | 擴展方向 |
|----------|----------|
| Introduction | 研究背景 → 產業趨勢 → 研究缺口 → 研究目的 |
| Literature Review | 分主題文獻 → 各研究方法對比 → Gap 識別 |
| Methodology | 實驗設計 → 設備規格 → 數據收集 → 分析方法 |
| Results | 圖表解釋 → 統計分析 → 趨勢描述 |
| Discussion | 結果解釋 → 與文獻對比 → 限制與建議 |

## 設定

```yaml
paper_evolution:
  # 文獻搜尋設定
  literature:
    max_results: 15
    min_relevance: 3  # 最少關鍵詞匹配數
    prefer_recent: 5  # 優先最近 5 年內文獻
    
  # APA 核查設定
  apa:
    style: "APA 7th"
    auto_fix: false  # 默认不自动修正，需用户确认
    
  # 擴展設定
  expansion:
    target_words: 5000  # 每章目標字數
    min_argument_count: 3  # 每段最少論點數
```
